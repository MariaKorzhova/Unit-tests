Объяснение того, какие сценарии покрыты тестами и почему выбрали именно эти сценарии:
Я сравнила два списка, в которых среднее значение первого списка больше среднего значения второго списка,
а также два списка, в которых среднее значение второго списка больше среднего значения второго списка.
И сравнила два списка, в которых средние значения равны. Выбрала эти сценарии для того, чтобы проверить,
что средние значения списков рассчитываются правильно, что метод по вычислению среднего значения верен, а также,
чтобы убедиться, что программа исправно сравнивает средние значения списков.
